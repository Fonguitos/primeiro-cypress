# Meu primeiro projeto em Cypress. Automação de Login.

## Esse projeto utiliza Cypres para automatizar uma feature de login e atualizção de perfil.

### Instação

```bash
npm install
```

> NOTA
>
> é necessário **node** instalado

### Como rodar
```bash
# Abra o Cypresss UI
npx cypress open

# Utilize esta linha de comando
npx cypress run
